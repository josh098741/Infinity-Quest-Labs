How to Use Navigation

The navigation has been automatically loaded! You can now:

  • Click the hamburger menu to open/close the sidebar
  • Use the search box in the sidebar
  • Navigate between pages using the sidebar links
  • Access the header navigation and profile area

Adding This to Your Pages

To add navigation to any of your existing pages, simply:

  1. Include navigation.css in your head section
  2. Include navigation-include.js before closing body tag
  3. Add the content area CSS (like the styles above)
  4. Wrap your content in a container with proper margins
